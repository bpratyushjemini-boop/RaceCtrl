# Roadmap - RaceCtrl Pro

Planned roadmap milestones for the RaceCtrl platform:

---

## 1. Live Webhook Ingestion (v1.1)
* Deploy active Pusher Channel listeners on the server to feed live sector intervals directly into the `/pro` dashboard without client polling.
* Add database triggers to synchronize saved Layout Profiles and API credentials across multiple devices.

---

## 2. Fantasy League Integration (v1.2)
* Fantasy Workspace profiles tailored for league analytics, driver value trackers, and league points progression charts.

---

## 3. Real-Time Weather Integration (v1.3)
* Direct airport radar loops feeding local precipitation forecasts to constructor timing pit walls.
