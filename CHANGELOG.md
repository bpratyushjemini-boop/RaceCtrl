# Changelog - RaceCtrl

All notable changes to the RaceCtrl platform are documented in this file.

---

## [1.0.0] - Stable Release Candidate
### Added
* **Custom Dashboard Builder**: Drag-and-drop widget toggles, panel resizing, layout duplicates, and layout profile slots with localStorage saving.
* **Advanced Analytics Studio**: Custom SVG pacing delta comparison charts and Report Exporters.
* **Developer API Center**: API key generation widgets, Webhook listener configurations, and interactive HTTP docs.
* **Automation Scheduler**: Task rules setting crons for briefings and alerts.
* **Presentation Mode Overlay**: Full-screen high-contrast telemetry timing overlay.

### Changed
* Simplified mobile navigation to exactly 5 tabs, routing secondary items to the `/more` portal.
* Reduced container padding to exactly 16px on mobile viewports for full-width cards.

---

## [0.6.0] - Motorsport Hub
### Added
* Support for WEC, Formula E, IndyCar, MotoGP, and 10 other championships.
* Unified Global schedule calendar with category filtering.
* Searchable technical glossary encyclopedia and rule timelines.

---

## [0.5.0] - AI Race Engineer
### Added
* Simulated undercut/overcut strategy calculators.
* AI weekend assistant briefs cards.
* Driver performance spider-web radars.
